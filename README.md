# 📚 Livraria

Projeto desenvolvido durante as aulas de **Programação Orientada a Objetos** utilizando Java.

O objetivo do projeto foi aplicar na prática conceitos de POO através da criação de um sistema simples de gerenciamento de uma livraria.

## 🚀 Funcionalidades

- Cadastro de livros;
- Visualização de livros disponíveis;
- Gerenciamento de produtos;
- Adição de livros ao carrinho de compras;
- Cálculo do valor da compra.

## 🛠️ Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (POO)
- Java Collections

## 📖 Conceitos praticados

- Classes e objetos;
- Encapsulamento;
- Herança;
- Polimorfismo;
- Manipulação de listas e mapas.
